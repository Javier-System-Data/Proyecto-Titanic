# Análisis de Supervivencia del Titanic - Visualización Avanzada

Este proyecto aplica técnicas de **Ingeniería de Datos** y **Estadística Descriptiva** para analizar los factores de supervivencia en el Titanic.

## 🛠️ Tecnologías utilizadas
* **Python** (Pandas, Numpy)
* **Visualización:** PtitPrince (Raincloud Plots), Seaborn, Matplotlib.
* **Entorno:** Google Colab.
